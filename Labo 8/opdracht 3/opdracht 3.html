Waarom staat de tekst in de blokken horizontaal gecentreerd? Dit komt door het css element” text-align” met als waarde “center”

Ga na dat een property vertical-align:center geen effect heeft. Waarom eigenlijk niet? Kan enkel gebruikt worden inline en table-cell elementen.

Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren? Je gebruikt element “line-height” en je geeft dit dezelfde waarde als je element “height”.

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren? Je kan jou display ook op table-cell gaan plaatsen. Dan kan je wel vertical align gebruiken.

In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang? Omdat deze meer onderaan het document staat.
